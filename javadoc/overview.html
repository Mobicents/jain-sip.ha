JAIN SIP HA Extension is an extension to existing implementation of NIST SIP Stack used in both Mobicents JAIN SLEE Resource Adaptor and Mobicents Sip Servlets. It adds the following capabilities to JAIN SIP applications using this extension :

  • Integration with Mobicents SIP Load Balancer
  • Replication of the dialog state when the dialog is in CONFIRMED state or when the application data on the dialog is modified
